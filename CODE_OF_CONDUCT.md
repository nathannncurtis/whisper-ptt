# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) v2.1.

## Standards

**Expected:**
- Respectful, constructive communication
- Accepting feedback gracefully
- Focusing on what's best for the project

**Unacceptable:**
- Harassment, insults, or personal attacks
- Trolling or deliberately inflammatory comments
- Publishing others' private information

## Enforcement

Instances of unacceptable behavior may be reported by opening an issue or contacting the maintainer directly. All complaints will be reviewed.
